# 🏏 Virat Kohli Career Dashboard (Power BI)

## 📊 Overview

This project analyzes the international cricket career of Virat Kohli using Power BI.

## 🔍 Key Metrics

* Total Matches: 516
* Total Runs: 24K+
* Total Centuries: 70
* Total Half Centuries: 192

## 📈 Features

* Format-wise performance analysis
* Year-wise trend visualization
* Milestone tracking (100s/50s)
* Interactive filters (Opponent, Year, Format, Stadium)
